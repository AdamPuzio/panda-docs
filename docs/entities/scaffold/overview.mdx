Scaffolds create things. They provide the ability to create new things with minimal effort. They can be run via code or as a command in a terminal.

* Inputs
  * `arguments`
  * `options`
  * `prompts`
* Outputs
  * `actions`

### Run Scaffold as Code
```js
import { ExampleScaffold } from './example-scaffold'

ExampleScaffold.build({})
```

### Run Scaffold as Command
 
```js
import { ExampleScaffold } from './example-scaffold'

ExampleScaffold.run({})
```
