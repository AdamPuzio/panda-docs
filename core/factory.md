# Factory

Factory is a Core class that specializes in scaffolding and Project management. 

Tasks such as parsing scaffold directories, retrieving scaffold files and templates, updating of project.json and package.json files, installing/uninstalling packages, etc. are all part if what Factory does best. 

## Usage

```js
const Factory = Core.Factory
```
