# Installation

## Standard Install

The best way to install Panda is by globally installing the CLI tool:

```bash
npm i -g panda-cli
```

Once you've installed it, you now have access to the `panda` command locally on your machine. 

```bash
# check the version
panda -v

# get a list of commands
panda
```
