# Frequently Asked Questions

## Why use `project:create` instead of building/scaffolding our own Project?

Panda provides a foundation to build things on, as well as tools to help you build them. We encourage you to build things in, around, and on top of Panda, but if you want to run them _with_ Panda, you're going to need certain things available that come from scaffolding. 

For example, if you want to build a website using `create-react-app`, you'll most likely want to build that _within_ a Panda Project if you want to ultimately run it using Panda. 

## What is the difference between building a Project and a Private Label?

Simply put, **Projects** are built and run. **Private Labels** are created so that _others_ can build their own Projects based on your implementation.
